Roombabot

Autonomous Mobile Robot · ROS 2

Overview

Roombabot is an autonomous mobile robot platform built on top of a Roomba base, running ROS 2 with a full SLAM, localization, and navigation stack. The platform is designed for research and development in autonomous navigation and robotic perception.

The robot combines hardware-level motor control with advanced software algorithms to achieve real-time autonomous navigation in indoor environments.

Technical Stack

  • Operating System: ROS 2 (Humble)
  • SLAM: SLAM Toolbox for real-time map building
  • Localization: Particle Filter-based localization
  • Sensor Fusion: EKF combining IMU and wheel odometry
  • Planning: NAV2 stack with global and local planning
  • Behavior: Behavior trees for mission planning
  • Languages: C++ and Python
  • Deployment: Docker containers for reproducibility
ROS 2 NAV2 SLAM Toolbox EKF Behavior Trees C++ Python Docker

Key Features

  • Real-time SLAM: Simultaneous Localization and Mapping using SLAM Toolbox
  • Autonomous Navigation: Path planning and obstacle avoidance using NAV2
  • Recovery Behaviors: Stuck detection and automatic recovery routines
  • Mission Planning: Behavior tree-based task execution and sequencing
  • Sensor Integration: LiDAR, IMU, and encoders with sensor fusion
  • Docker Deployment: Containerized environment for easy setup and reproducibility

Current Development

The project is actively being developed with focus on:

  • Learning behavior trees for complex mission planning
  • Parameter tuning for better real-world navigation robustness
  • Debugging encoder issues on the real-world robot

BFS Path Planning

Breadth-First Search (BFS) pathfinding demo. This algorithm explores the environment systematically to find the shortest path from the robot's current position to the goal.

RRT Path Planning

Rapidly-exploring Random Tree (RRT) pathfinding demo. This algorithm efficiently explores high-dimensional spaces by building a tree of random samples, allowing for rapid planning even in complex environments.

Astar Path With PD Local Planner

Comparison of two PD local planner implementations with A* pathfinding. Watch both approaches side-by-side to see how different tuning parameters affect navigation behavior.

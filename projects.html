Roombabot

Autonomous Mobile Robot · ROS 2

Autonomous mobile robot platform built on top of a Roomba base, running ROS 2 with full SLAM, localization, and navigation stack.

Actively developing behavior trees for mission planning, recovery behaviors for stuck detection, and navigation parameter tuning for real-world robustness.

  • SLAM Toolbox for real-time map building
  • EKF sensor fusion (IMU + wheel odometry)
  • NAV2 global & local planning
  • Docker-based deployment for reproducibility

Reinforcement Learning for Autonomous Highway Driving

Machine Learning · Python · Highway-Env

This project implements and compares multiple reinforcement learning algorithms for autonomous vehicle control in highway driving scenarios. The goal is to train agents that can safely navigate complex traffic environments, make optimal driving decisions, and achieve efficient lane changes and speed control.

Using the Highway-Env simulation environment, we evaluated six different reinforcement learning algorithms including A2C, PPO, DQN, Dueling DQN, DDPG, and APPO, comparing their performance across metrics like safety, efficiency, and learning stability.

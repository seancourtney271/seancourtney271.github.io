Reinforcement Learning for Autonomous Highway Driving

Machine Learning · Python · Highway-Env

Overview

This project implements and compares multiple reinforcement learning algorithms for autonomous vehicle control in highway driving scenarios. The goal is to train agents that can safely navigate complex traffic environments, make optimal driving decisions, and achieve efficient lane changes and speed control.

Using the Highway-Env simulation environment, we evaluated six different reinforcement learning algorithms: Advantage Actor-Critic (A2C), Proximal Policy Optimization (PPO), Deep Q-Network (DQN), Soft Actor-Critic (SAC), Twin Delayed Deep Deterministic Policy Gradient (TD3), and Advantage Proximal Policy Optimization (APPO).

The project demonstrates the application of modern RL techniques to real-world autonomous driving challenges, comparing algorithm performance across metrics like safety, efficiency, and learning stability.

Technical Stack

  • Python: Primary programming language for implementation
  • Stable-Baselines3: Reinforcement learning library providing algorithm implementations
  • Highway-Env: Open-source environment for autonomous driving research
  • PyTorch: Deep learning framework for neural network training

Algorithms Overview

The project evaluates six reinforcement learning algorithms, each with different approaches to learning optimal policies:

  • A2C (Advantage Actor-Critic): Synchronous version of A3C that uses advantage function estimation to reduce variance in policy gradient updates
  • PPO (Proximal Policy Optimization): Trust-region method that constrains policy updates to prevent destructive changes while maintaining learning efficiency
  • DQN (Deep Q-Network): Value-based method using deep neural networks to approximate Q-values, with experience replay and target networks for stability
  • Dueling DQN: Extension of DQN that separately estimates state values and action advantages, improving learning efficiency in complex environments
  • DDPG (Deep Deterministic Policy Gradient): Actor-critic method for continuous action spaces, combining deterministic policy gradients with deep Q-learning
  • APPO (Advantage Proximal Policy Optimization): Advanced version of PPO with improved advantage estimation and better exploration strategies

Status

This project is complete. All six reinforcement learning algorithms have been successfully implemented, trained, and evaluated on the highway driving task. The results demonstrate varying performance characteristics across different RL approaches, with some algorithms showing better safety records while others achieve higher efficiency.

The project includes comprehensive documentation, performance analysis, and visual demonstrations of the trained agents in action.
